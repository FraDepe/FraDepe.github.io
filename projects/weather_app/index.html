Introduction

This application was my internship project for my bachelor's thesis. It's a weather app with a multimodal interface so that it can be used also by visually impaired and blind people.

Features

  • Audio guide to explain how to use the application
  • Voice prompt to execute a request specifing a location and optionally a time specification
  • Two different modes. One for sighted people and one for visually impaired and blind people
  • Error handler with voice alert
  • Multimodal interface
  • Voice response to provide requested information
